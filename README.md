# Webpack setup with TS/Pug/Less

### Install dependencies
```
npm install
```

### Install typescript definitions manager (optional)
```
npm install typings
typings instal
```

### Build release
```
npm run deploy
```

### Build and watch for changes
```
npm run watch
```

### Build, watch for changes and run dev-server on 8080
```
npm run server
```
